## P575 Skew Binary
#### Daniel 'June' Portillo
##### Discription: 

This program takes in an input in 'Skew Binary'
,where '100' would be 1 * 2^2 is enstead 1* (2^ (n+1) -1) and where
the least significant non-zero digit may be 2, and converts it
into decimal.
